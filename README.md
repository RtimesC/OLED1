# OLED1

一个基于 STM32 的 OLED 显示屏驱动项目。

## 项目简介

本项目实现了 OLED 显示屏的驱动程序，使用 Keil µVision 开发环境。

## 项目结构

```
OLED1/
├── Hardware/       # 硬件驱动代码
├── Library/        # 库文件
├── System/         # 系统文件
├── User/           # 用户应用代码
├── Start/          # 启动文件
├── Objects/        # 编译生成的目标文件
└── Listings/       # 编译生成的列表文件
```

## 开发环境

- IDE: Keil µVision
- 语言: C (89.8%), Assembly (5.7%)
- 目标芯片: STM32 系列

## 使用说明

1. 使用 Keil µVision 打开 `Project.uvprojx` 文件
2. 编译项目
3. 下载到目标板
